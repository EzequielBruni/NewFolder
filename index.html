NewFolder

A design system by Ezequiel Bruni, for Ezequiel Bruni.

Introduction

NewFolder is a lightweight, minimalist design system / CSS framework, created to give users an easy reading experience. It was made to be used with blogs, wikis, and knowledge-base style sites.

But really, it's a hodge-podge of styles that I've either written myself, or borrowed from elsewhere over the years. It's meant to be easy to customize in terms of color and typography, though I'll be sticking with the type I am using here.

All layouts must be created by the user. I like to use CSS Grid for that, myself.

Requirements

A working knowledge of HTML, CSS, and SASS. A willingness to interpret my somewhat minimal code-commenting style. An inexplicable desire to use this hyper-specific design system/SASS framework that I made for my own personal use.

Colors

Colors are set up via SASS variables, to make it easy to change the whole look of your design system at once. Primary text colors are set with $dark-text and $light-text.

I think of the main color variables as more of a starting point, and make small adjustments via SASS darken/lighten operations as needed.

$secondary-color
