NewFolder

A design system by Ezequiel Bruni, for Ezequiel Bruni.

NewFolder is a lightweight, minimalist design system / CSS framework, created to give users an easy reading experience. It was made to be used with blogs, wikis, and knowledge-base style sites.

Colors

Colors are set up via SASS variables, to make it easy to change the whole look of your design system at once. Primary text colors are set with $dark-text and $light-text.
